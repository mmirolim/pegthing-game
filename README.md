# pegthing

Pegthing game from Functional Programming chapter of Clojure for the Brave and True book.

## Usage

	$ lein run
or

	$ java -jar pegthing-0.1.0-standalone.jar [args]

## Examples

	Hello, let's play Peg thing!
	How many rows? [5]

	Here's your board
		  a0
		 b0 c0
	   d0 e0 f0
	  g0 h0 i0 j0
	k0 l0 m0 n0 o0
	Remove which peg? [e]

...
